### 建立双周同步机制的预期目的：

1. 帮助东亚和澳洲时区的小伙伴们相互之间交流进展和需求。
2. 用中文视频和语音交流。
3. 上班时间交流。
4. 安排能熬夜的年轻人去参加RVI的会议并在双周同步会上总结（这次轮到老年人不讲武德😄）。
